# The Paper Dream.

# Live link:
    "https://thepaperdream.netlify.app/"

## Features of the e-commerce app (v1.0)

 - Landing Page
 - Products Listing Page
 - Wishlist Page
 - Cart Management
 - Authentication Pages - Login and Sign Up
 - Filters Panel
